# Barbearia The Blinders 🪒✂️

Landing page completa da Barbearia The Blinders - um projeto educacional de desenvolvimento web!

## 🎨 **Sobre o Projeto**

Este é um site responsivo moderno para uma barbearia com tema old school, desenvolvido com as melhores práticas de HTML5, CSS3 e design responsivo.

### ✨ **Características Principais:**
- **Design Dark**: Tema escuro com tons de preto, cinza chumbo e dourado envelhecido
- **Totalmente Responsivo**: Funciona perfeitamente em mobile, tablet e desktop
- **Animações Suaves**: Efeitos hover e transições elegantes
- **Integração WhatsApp**: Botão direto para contato
- **Agendamento Online**: Integração com Booksy
- **Galeria de Fotos**: Showcase profissional dos serviços

## 📁 **Estrutura do Projeto**

```
BarbeariaBlinders-/
├── index.html          # Página principal do site
├── styles.css          # Estilos CSS organizados e comentados
├── IMAGENS/           # Pasta com todas as imagens do site
│   ├── FAIXADA.jpeg
│   ├── DECORACAO.jpeg
│   ├── DONO.jpeg
│   └── ... (mais imagens)
├── Img/               # Backup das imagens originais
└── README.md          # Este arquivo
```

## 🚀 **Como Usar**

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Lima01-01/BarbeariaBlinders-.git
   ```

2. **Abra o arquivo `index.html` no navegador**

3. **Pronto!** O site está funcionando localmente

## 🎓 **Valor Educacional**

Este projeto foi desenvolvido pensando em estudantes de desenvolvimento web:

### 📚 **O que você vai aprender:**
- **HTML5 Semântico**: Estrutura organizada com header, nav, sections, etc.
- **CSS3 Moderno**: Flexbox, Grid, animações, responsividade
- **Design Responsivo**: Media queries e layouts adaptáveis
- **Boas Práticas**: Código comentado e organizado
- **SEO Básico**: Meta tags e estrutura semântica

### 💡 **Comentários Especiais:**
O arquivo `styles.css` contém comentários **super descontraídos e informais** explicando cada parte do código de forma amigável para estudantes!

## 🛠️ **Tecnologias Utilizadas**

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com comentários educacionais
- **Google Fonts**: Inter e Playfair Display
- **Tailwind CSS**: Framework CSS via CDN
- **Git**: Controle de versão

## 📱 **Seções do Site**

1. **Header**: Navegação fixa com logo e menu
2. **Hero**: Apresentação principal com chamada para ação
3. **Sobre**: História da barbearia
4. **Serviços**: Corte, barba e pacotes com preços
5. **Galeria**: Fotos profissionais da barbearia
6. **Contato**: Informações, horário e botões de contato
7. **Footer**: Direitos autorais
